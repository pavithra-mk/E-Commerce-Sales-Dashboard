# 🛒 E-Commerce Sales Dashboard | Power BI

## 📌 Project Overview
This project is an interactive **E-Commerce Sales Dashboard** built using **Microsoft Power BI**. The dashboard provides valuable insights into sales performance, profit, customer behavior, and product trends. It helps businesses monitor key performance indicators (KPIs) and make data-driven decisions.

---

## 🎯 Objectives
- Analyze overall sales performance.
- Track profit and revenue.
- Identify top-performing products and categories.
- Monitor sales by region/state.
- Understand customer purchasing patterns.
- Provide interactive filtering for better analysis.

---

## 📊 Dashboard Features

### KPI Cards
- 💰 Total Sales
- 📦 Total Quantity Sold
- 💵 Total Profit

### Visualizations
- Sales by Category
- Sales by Sub-Category
- Monthly Sales Trend
- Sales by State/Region (Map)
- Profit Analysis
- Product Performance
- Category-wise Sales Distribution (Pie Chart)

### Interactive Filters
- Category
- State/Region
- Product
- Date
